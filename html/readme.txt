🌤️ **Weather Info Website** 🌤️

I'm excited to share my latest project: a  Weather Info Website built using HTML, CSS, and JavaScript. This project was a fantastic opportunity to enhance my front-end development skills and work with real-time data fetching using APIs.

🔧 **Technologies Used:**
- HTML, CSS, JavaScript
- OpenWeatherMap API for fetching weather data
- OpenWeatherMap API for city name suggestions
- GitHub for version control

✨ **Key Features:**
- **Real-Time Weather Data**: Get the latest temperature, humidity, and wind speed for any city.
- **City Name Suggestions**: Enjoy enhanced user experience with dynamic city name suggestions.
- **Dynamic Content Updates**: Experience efficient updates without page reloads.

🎓 **What I Learned:**
- Mastered asynchronous programming and API integration.
- Improved my skills in JavaScript for dynamic content manipulation.

- Developed a clean, modular codebase for better maintainability and scalability.
